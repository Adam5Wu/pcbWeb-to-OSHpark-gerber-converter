This is quick python script to convert the gerber files generated by the pcb 
design tool pcbWeb (http://www.pcbweb.com/) intogerber files that can be 
understood by the pcb batch order service OSHPark (https://oshpark.com/)

Requirements:
This expects that you are running python 2.7

It should run on pretty much any operating system

Instructions:
Run the python script using your prefered method and obey the instructions
that it gives you.

current limmitations:
currently the script will only work for 2 layer boards. It also ignores the non
plated holes file generated by pcbWeb.
